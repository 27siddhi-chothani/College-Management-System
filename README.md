# College Management System (SQL Project)

## 📌 Objective
Manage student academic records and attendance using SQL.

## 📊 Tables
- Students
- Courses
- Faculty
- Attendance
- Results

## 🔍 Key Analysis
- Top-performing students
- Attendance percentage
- Course enrollment analysis
- Faculty workload

## 🛠 Concepts Used
- Views
- Nested Queries
- Constraints
- Joins
- Aggregate Functions

## 📈 Insights
- Some students have attendance below 75%
- Faculty workload varies by department
- Top students consistently score high across courses
